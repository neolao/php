Class RouteRegexp

Route with a regexp as pattern

Example of a configuration:

$config = [
     'pattern'       => '/^\/account\/([a-zA-Z]+)\/order\/([0-9]+)\/pdf$/',
     'map'           => ['name', 'orderId'],
     'controller'    => 'foo',
     'action'        => 'bar',
     'reverse'       => '/account/%s/order/%d/pdf'
];
